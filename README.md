# anki

Минимальное веб‑приложение «как AnkiWeb» (пока без сервера и без синхронизации), с UI в стиле проекта `alias`.

## Запуск

Важно: открывайте через HTTP‑сервер (не `file://`), иначе некоторые браузеры могут ограничивать работу PWA/manifest.

### Вариант 1: Python

`python3 -m http.server 5173`

Откройте: `http://localhost:5173`

### Вариант 2: Node.js

`npx serve .`

## Что уже есть

- Колоды: создание/переименование/удаление
- Карточки (Basic): front/back + теги
- Повторение: flip + кнопки Again/Hard/Good/Easy (упрощённый SM-2‑подобный алгоритм)
- Браузер: поиск по front/back/tags + редактирование
- Статистика: today + due/new, лимиты new/day и reviews/day
- Импорт/экспорт в JSON
